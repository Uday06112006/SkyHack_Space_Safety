# Space-Safety: Orientation-Agnostic Object Detection
Project for SkyHack 2.0 | Team Binary Brains

## The Problem
In microgravity, safety equipment (fire extinguishers, oxygen tanks) floats at random 360-degree angles. Standard AI often fails to recognize objects when they are upside down.

## Our Solution
Our project solves the 'Orientation-Dilemma' problem in microgravity environments with the creation of a gravity-neutral object detection system for astronaut safety items. Utilizing the YOLOv8 model, along with synthetic data creation through the Duality AI Falcon platform/Roboflow, our model has been trained to identify essential safety items such as fire exttinguishers,Space suit,helmet,wrench at any rotational position of 360 degrees.
